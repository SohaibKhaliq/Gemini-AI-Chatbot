## 📖 Overview

Minimal Python script demonstrating the Google Gemini 2.0 Flash API using the `google-genai` SDK.

## 🚀 Quick Start

```bash
pip install -r requirements.txt
# Edit chatbot.py — replace YOUR-API-KEY with your Gemini API key
python chatbot.py
```
